# Minecraft Server Invite

DM me on LinkedIn to join and see my SMP Minecraft server — I build offices I worked on IRL inside the world. 

No mods needed to join, all you need is minecraft version 26.1.2. All the mods I have created and are being used on the server are server-side, and you can use them without installing anything!

---

## Mods in this repository

- **No Elytra Mod**: Prevents or enforces restrictions on Elytra usage. Includes a command (`ElytraCommand`), configuration (`NoElytraConfig` / `ConfigManager`), and enforcement logic (`ElytraEnforcer`) plus client and mixin resources. See the `No Elytra Mod` folder for implementation details.
- **Server Auto Click**: lets players use timed auto-attack and auto-consume commands without installing the mod client-side. Includes a /autoattack and /autoconsume command. See the `Server Auto Click` folder for implementation details.
